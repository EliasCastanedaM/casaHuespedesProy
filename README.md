# Casa Huéspedes Pimentel

Sistema web para Casa Huéspedes Pimentel.

## Objetivo del sistema

Permitir que los clientes puedan ver habitaciones, consultar servicios, reservar por fechas, pagar online y que el administrador pueda gestionar reservas, clientes, imágenes y estadísticas.

## Módulos principales

- Web pública
- Habitaciones
- Reservas por fecha
- Pago online
- Clientes frecuentes
- Panel administrativo
- Carga de imágenes
- Estadísticas
- Consultas
- Página turística de Lambayeque

## Tecnologías

- React + Vite
- Tailwind CSS
- Node.js + Express
- PostgreSQL
- Cloudinary
- JWT
